形容詞【けい・よう・し】・形容動詞【けい・よう・どう・し】

Properties of Adjectives

Now that we can connect two nouns together in various ways using particles, we want to describe our nouns with adjectives. An adjective can directly modify a noun that immediately follows it. It can also be connected in the same way we did with nouns using particles. All adjectives fall under two categories: adjectives end with 「い」 and adjectives end widh 「だ」.

The 形容詞【けい・よう・し】

All 形容詞【けい・よう・し】 always end in the Hiragana character: 「い」. Unlike 形容動詞【けい・よう・どう・し】, you do not need to add 「な」 to directly modify a noun with an 形容詞【けい・よう・し】.

Vocabulary

1. あの - that
2. 食べ物【た・べ・もの】- food
3. 素晴らしい【すば・らしい】 - wonderful
4. 高い【たか・い】- high; tall; expensive
5. ビル - building
6. 値段【ね・だん】- price
7. レストラン - restaurant
8. あまり・あんまり - not that much
9. もっと - much more

Below are the rules for conjugating 形容詞【けい・よう・し】. Notice that the rule for conjugating to negative past tense is the same as the rulse for the past tense.

Conjugation rules for 形容詞【けい・よう・し】

高 い Base
たか くない Negative
たか くて Connective
たか くても Contrastive
たか かった Past-tense
たか かったら Predictive I
たか ければ Predictive II
美味し い Base
おい・し くない Negative
おい・し くて Connective
おい・し くても Contrastive
おい・し かった Past-tense
おい・し かったら Predictive I
おい・し ければ Predictive II

Examples

1. 高いビル。
Tall building.

2. 高くないビル。
Not tall building.

3. 高かったビル。
Building that was tall.

5. 高くなかったビル。
Building that was not tall.
6. 高くて素晴らしいビル
Tall and wonderful building.

7. 高くても素晴らしくないビル
Tall but not wonderfule building

In this example, the descriptive clause 「高い」 is directly modifying 「ビル」.

Examples

1. あのビルは高い。
That building is tall.

2. あのビルは高くない。
That building is not tall.

3. あのビルは高かった。
That building was tall.

4. あのビルは高くなかった。
That building was not tall.

5. あのビルは高くて素晴らしい。
That building is tall and wonderful

6. あのビルは高くても、あまり素晴らしくない。
That building is tall but not that much wonderful.

7. あのビルはあまり高くなくても、素晴らい。
That building is not that much tall but wonderful.

8. あのビルは高かったら、もっと素晴らしい。
If that building is tall, it will be much more wonderful.

9. あのビルは高ければ、もっと素晴らしい。
If that building is tall, it will be much more wonderful.

In this example, the descriptive clause 「高い」 describes about「ビル」.

An annoying exception

Vocabulary

1. 値段【ね・だん】- price
4. 彼【かれ】- he; boyfriend

5. 良い【い・い / よ・い】 - good

5. 格好いい【かっ・こう・いい】 - cool; handsome

There is one 形容詞【けい・よう・し】 meaning "good" that acts slightly differently from all other 形容詞【けい・よう・し】. This is a classic case of how learning Japanese is harder for beginners because the most common and useful words also have the most exceptions. The word for "good" was originally 「よい (良い)」 . However, with time, it soon became 「いい」. When it is written in Kanji, it is usually read as 「よい」 so 「いい」 is almost always Hiragana. That's all fine and good. Unfortunately, all the conjugations are still derived from 「よい」 and not 「いい」. This is shown in the next table.

Another adjective that acts like this is 「かっこいい」 because it is an abbreviated version of two words merged together: 「格好」 and 「いい」. Since it uses the same 「いい」, you need to use the same conjugations.

い い Base
よ くない Negative
よ くて Connective
よ くても Contrastive
よ かった Past-tense
よ かったら Predictive I
よ ければ Predictive II
格好い い Base
かっこう・よ くない Negative
かっこう・よ くて Connective
かっこう・よ くても Contrastive
かっこう・よ かった Past-tense
かっこう・よ かったら Predictive I
かっこう・よ ければ Predictive II

Take care to make all the conjugations from 「よい」 not 「いい」.

Examples

1. 値段があんまりよくない。
Price isn't very good.

2. 彼はかっこよかった!
He looked really cool!

The 形容動詞【けい・よう・どう・し】

Vocabulary

1. 人【ひと】- person
2. 友達【とも・だち】- friend
3. 親切だ【しん・せつ・だ】(na-adj) - kind
4. 魚【さかな】- fish
5. 好きだ【す・きだ】(na-adj) - likable; desirable
6. 肉【にく】- meat
7. 野菜【や・さい】- vegetables

The 形容動詞【けい・よう・どう・し】 is very simple to learn because it acts essentially like a noun. All the conjugation rules for both nouns and 形容動詞【けい・よう・どう・し】 are the same. One main difference is that a 形容動詞【けい・よう・どう・し】 can directly modify a noun following it by sticking 「な」 between the adjective and noun.

Examples

1. 友達は親切だ。
Friend is kind.

2. 友達は親切な人だ。
Friend is kind person.

As shown by the following examples, the conjugation rules for na-adjectives are the same as nouns. Below are the rules for conjugating 形容動詞【けい・よう・どう・し】.

好き だ Base
す・き な Modifying
す・き ではない Negative
す・き で Connective
す・き でも Contrastive I
す・き だが Contrastive II
す・き だった Past-tense
す・き だったら Predictive I
す・き なら Predictive II
親切 だ Base
しん・せつ な Modifying
しん・せつ ではない Negative
しん・せつ で Connective
しん・せつ でも Contrastive
しん・せつ だった Past-tense
しん・せつ だったら Predictive I
しん・せつ なら Predictive II

Examples

1. 彼は親切な人が好きだ
He likes kind person

2. あの人は親切ではない。
That building is not tall.

3. あの人は好きだった。
That building was tall.

4. あの人は親切ではなかった。
That building was not tall.

5. あの人は親切で格好いい。
That building is tall and wonderful

6. あの人は親切でも、格好よくない。
That building is tall but not that much wonderful.

7. あの人は好きだが、親切じゃない。
That building is tall but not that much wonderful.

8. あの人は親切だったら、好きだ
That building is not that much tall but wonderful.

9. あの人は親切なら, 好きだ
If that building is tall, it will be much more wonderful.
